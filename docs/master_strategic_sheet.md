# Master Strategic Sheet (The Swarm Constitution)

## Section 1: The North Star
**Goal:** A commerce-native design tool that beats Canva on Frictionless Personalization.
**Core Metrics:** <30s to first design success; 60fps canvas performance; 100% print accuracy.

## Section 2: Technical Constraints
**Stack:** Next.js, Fabric.js v6 (TS), Tailwind, Radix UI.
**Backend:** OpenMage API, PostgreSQL, S3 Asset Storage.
**Architecture:** Headless, Contract-First (JSON).

## Section 3: Interaction Non-Negotiables
**Kinetic UI:** All transitions <200ms.
**Smart Snapping:** Magnetic center/edge guides with "Pink Line" visuals.
**Production Aware:** Real-time "Safe Zone" and "DPI" warnings.
